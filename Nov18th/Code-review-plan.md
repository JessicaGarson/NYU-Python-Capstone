# How will we handle code reviews?

## Values 
- Be nice 
- Be patient
- Stay focused 

## Checklist
- [ ] Does the code work? 
- [ ] Readability
- [ ] Can this be done better
- [ ] Does the code repeat itself 
- [ ] Is there anything you'd like a second opinion on?
- [ ] Security concerns (sql injection, exposed passwords/api keys, etc.)
